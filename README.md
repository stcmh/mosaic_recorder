# Mosaic Recorder

이 프로그림은 YOLO를 사용하여 동영상에서 사람 얼굴을 인식하고 모자이크한다. 또한 추가적인 입력을 통해 동영상 정지 및 시간 이동등 기본적인 동영상 플레이어 역할을 제공한다. 

## How to use

코드 실행 후 동영상을 선택, space를 통해 동영상의 녹화, 클릭 및 드래그를 통해서 사용가능하다.

## Requirements

```bash
pip install ultralytics opencv-python tkinter
```
